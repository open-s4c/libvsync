#  vsync
_vatomic is a header library of atomics operations, supporting mainstream architectures: ARMv7, ARMv8 (AArch32 and AArch64), RISC-V, and x86_64._ 

The memory ordering guarantees provided by the atomic interface are formally described in the VSync Memory Model (VMM) file.
